Exercise 5 – String Analyzer Tool(Strings + Collections + Exception Handling)
Problem Statement:
  Analyze a list of user-entered strings.
Requirements:
    Accept multiple strings from user
    Store them in List<String>
Operations:
    Reverse each string
    Count vowels in each string
    Find duplicate strings
    Throw custom exception if empty string entered
    Use Streams to filter strings with length > 5
Concepts Used
✅ Strings
✅ Collections
✅ Custom Exception
✅ Java 8 Streams
